# Smart Task Manager

A console-based task management application built using C# and .NET.

## Features
- Add Tasks
- Delete Tasks
- Edit Tasks
- Search Tasks
- Priority Sorting
- Statistics Dashboard
- File Saving
- Categories & Due Dates

## Technologies Used
- C#
- .NET
- Console Application

## Future Improvements
- GUI Version
- Database Integration
- Notifications
- ASP.NET Web App
